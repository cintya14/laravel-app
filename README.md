# laravel-app

Tienda online desarrollada en Laravel.

Este proyecto utiliza Laravel como framework principal para la creación de una tienda virtual orientada a microempresas.

## Tecnologías usadas
- Laravel 10
- PHP 8.2
- MySQL
- Tailwind CSS
- Docker (opcional)

## Descripción
Aplicación web para la gestión de productos, ventas, usuarios y reportes de una tienda online.

## Instalación
1. Clonar el repositorio.
2. Ejecutar `composer install`.
3. Configurar el archivo `.env`.
4. Ejecutar migraciones con `php artisan migrate`.
5. Levantar el servidor con `php artisan serve`.
